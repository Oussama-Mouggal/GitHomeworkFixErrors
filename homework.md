## 4. What happens when you `fork` a repository?

Forking a repository means you can make any changes you want to the code without affecting the original project. 

## 5. What happens when you clone a repository?

you download a complete copy of a remote project, including all its files, commit history, and all branches, to your local machine
## 6. What is a Pull Request?

 is a formal proposal in a software development project, often using distributed version control systems like Git and hosting platforms like GitHub, to merge code changes from one branch into another, usually into the main project repository.
